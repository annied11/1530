# 1530 Project
A simple web app for University of Pittsburgh club events.

## How to Run
1. Fork and clone the repo  
2. Open `data/events_sample.json` to view mock events (TBD)
3. Run the backend file to serve `/events` (TBD)
4. Open the frontend page to see the event list (TBD)

## Structure
    1530/
    ├── frontend/         # Simple page that lists events
    ├── backend/          # Backend file that serves /events
    ├── data/             # Dataset with club events
    ├── docs/             # Notes or diagrams for each sprint
    └── README.md         # Project overview and setup instructions
